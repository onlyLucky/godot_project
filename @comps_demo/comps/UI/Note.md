## UI 动效

**各个引擎UI系统**
- UE4 控件蓝图
- Unity NGUI UGUI
- Godot UI系统
- AE 仅供参考锻炼美感

**容易陷入误区**
- 【误区】将3D场景捕获2D屏幕 （其实还可以将UI放入到3D场景中  eg: 王者荣耀 英雄展示  原神 角色页面 圣遗物升级页面）
- 【误区】通过Shader实现轨迹运动或者其他动效（其实还可以使用建模转UV 使用shader实现）不要完全依赖shader,要搭配建模UV， 时间轴帧动画

**UI 动效常用的制作方法**
1. AE制作序列帧，播放序列帧
2. 引擎制作粒子特效通过摄像头捕捉到ui上
3. 直接在3D场景中制作UI
4. 绘制模型到UI上
